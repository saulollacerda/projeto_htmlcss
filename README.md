# Projeto Final HTML5 - Wiki do São Paulo Futebol Clube

## Sobre o projeto

Este repositório contém o projeto final da disciplina de **Programação Web** (UNIFSA), desenvolvido com **HTML5 puro**.

A proposta foi criar um site utilizando **somente HTML**, com foco em:

- estrutura correta do documento
- uso de **tags semânticas**
- organização do conteúdo
- navegação entre seções

O tema escolhido foi uma **wiki temática sobre o São Paulo Futebol Clube (SPFC)**.

## Tema da Wiki

A página apresenta informações organizadas sobre o clube, incluindo:

- história
- títulos e conquistas
- ídolos
- estádio
- curiosidades
- referências externas

Além disso, o projeto também inclui um **formulário simples** (item bônus da atividade).

## Arquivo principal

- `index.html` — página principal do projeto

## Requisitos da atividade atendidos

### 1) Estrutura básica do HTML5
- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<title>`
- `<meta charset>`
- `<body>`

### 2) Tags semânticas
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<aside>`
- `<footer>`

### 3) Conteúdo textual
- títulos (`<h1>` até `<h4>`)
- parágrafos (`<p>`)
- citação (`<blockquote>`)

### 4) Listas
- lista ordenada (`<ol>`)
- lista não ordenada (`<ul>`)

### 5) Tabela
Tabela estruturada com:
- `<table>`
- `<thead>`
- `<tbody>`
- `<tr>`
- `<th>`
- `<td>`

### 6) Mídia
- imagem com `<img>` e atributo `alt`
- conteúdo externo com `<iframe>` (vídeo)

### 7) Navegação
- links internos (âncoras no menu)
- links externos com `target="_blank"`

### 8) Bônus (opcional)
- formulário com `<form>`, `<label>`, `<input>` e `<button>`

## Como abrir o projeto

### Localmente
Basta abrir o arquivo `index.html` em qualquer navegador.

### Publicado no GitHub Pages
Após subir o projeto para o GitHub, é possível publicar pelo **GitHub Pages**:

1. Acesse **Settings**
2. Vá em **Pages**
3. Selecione a branch principal (`main`)
4. Salve e copie o link gerado
